# Web-Fetch 核心功能规划

## 项目概述

Web-Fetch 是一个面向 AI 的网页数据采集服务，将任意网站转换为 LLM 可用的结构化数据。

## 核心功能 (全部完成 ✅)

### 1. 网页抓取 (Scrape) ✅
- 单 URL 转换为 Markdown / HTML / JSON
- 支持代理访问国外网站
- 输出格式：markdown, html, links

### 2. 网站爬取 (Crawl) ✅
- 从起始 URL 递归抓取整个网站
- 可配置最大深度 (maxDepth)
- 自动去重，避免重复抓取

### 3. 结构化提取 (Extract) ✅
- 自动提取标题、描述
- 自动提取链接列表
- 无需预定义 Schema

### 4. 搜索 (Search) ✅
- Bing 搜索集成
- 返回搜索结果列表
- 支持结果数量控制

### 5. 代理支持 ✅
- 自动使用 HTTP_PROXY 环境变量
- 默认代理: http://127.0.0.1:7890
- 支持访问 BBC/Reuters 等国外网站

## API 设计

### POST /scrape
```json
{
  "url": "https://example.com",
  "formats": ["markdown", "html", "links"],
  "options": {
    "timeout": 30000
  }
}
```

### POST /crawl
```json
{
  "url": "https://example.com",
  "maxDepth": 2,
  "limit": 20
}
```

### POST /search
```json
{
  "query": "AI news",
  "limit": 10
}
```

### POST /extract
```json
{
  "url": "https://example.com"
}
```

## 输出格式

### Markdown
干净的 Markdown 内容，适合 LLM 处理

### JSON
结构化数据

### Metadata
- title
- description  
- sourceURL
- statusCode

## 技术实现

- 使用 curl 作为 HTTP 客户端
- 支持代理转发
- 轻量级，无外部依赖
