# Web-Fetch 核心功能规划

## 项目概述

Web-Fetch 是一个面向 AI 的网页数据采集服务，将任意网站转换为 LLM 可用的结构化数据。

## 核心功能

### 1. 网页抓取 (Scrape)
- 单 URL 转换为 Markdown / HTML / JSON
- 支持 JavaScript 渲染（动态页面）
- 输出格式：markdown, html, json, screenshot, links

### 2. 网站爬取 (Crawl)
- 从起始 URL 递归抓取整个网站
- 可配置最大深度 (maxDepth)
- 自动去重，避免重复抓取

### 3. 结构化提取 (Extract)
- 基于 Pydantic Schema 的结构化数据提取
- AI 智能提取（无需预定义 Schema）
- 支持自定义提取规则

### 4. 搜索 (Search)
- Web 搜索能力
- 返回搜索结果 + 完整页面内容

### 5. 代理支持
- 代理轮换
- 反检测策略
- 请求频率控制

### 6. 浏览器操作
- Click / Type / Scroll / Wait
- 登录认证后抓取
- 截图功能

## 技术栈

- **核心框架**: Go (Hertz) + Playwright
- **前端**: 待定
- **存储**: SQLite / 文件存储

## API 设计

### POST /scrape
```json
{
  "url": "https://example.com",
  "formats": ["markdown", "html"],
  "options": {
    "waitFor": ".content",
    "timeout": 30000
  }
}
```

### POST /crawl
```json
{
  "url": "https://example.com",
  "maxDepth": 3,
  "concurrency": 5
}
```

### POST /extract
```json
{
  "url": "https://example.com",
  "schema": {
    "type": "object",
    "properties": {
      "title": {"type": "string"},
      "price": {"type": "number"}
    }
  }
}
```

## 输出格式

### Markdown
干净的 Markdown 内容，适合 LLM 处理

### JSON
结构化数据，支持 Schema 验证

### Metadata
- title
- description  
- sourceURL
- statusCode
- crawlDepth
